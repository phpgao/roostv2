roost
Terminal TUI Session Manager for AI Coding Assistants
CodeBuddy Claude Gemini Codex Copilot OpenCode

Install

brew install phpgao/tap/roost go install github.com/phpgao/roost@latest

Requires Go 1.26+

Features

◉ Splash screen with animated logo
◉ Project + session list views
◉ Vim-style keyboard navigation
◉ Session detail & manual commands
◉ Batch select / delete
◉ Search with real-time filtering
◉ Platform filter (Tab)
◉ Custom agent support
◉ Responsive layout
◉ 1-minute auto-refresh
◉ CLI subcommands
◉ Cyberpunk dark theme

Keyboard Shortcuts

?/k Move up / down
g / G First / last item
Enter Open project / Resume session
n New session (agent picker)
Esc Back / Double-tap to quit
/ Search
d Delete current item
x Delete project (session view)
Space Batch select / Toggle
D Batch delete selected
Tab Cycle platform filter
r Refresh
? Full keybinding list
q / Ctrl+C Quit

Quick Start

$ roost              # Launch interactive TUI
$ roost list         # List all projects
$ roost list -t session -p /path/to/repo   # List sessions
$ roost resume -s <session-id>             # Resume a session
$ roost delete -s <session-id>             # Delete a session

Configuration

Edit ~/.roost/roost.yaml

# resume mode: replace (default) or suspend
resume_mode: suspend

platforms:
  codebuddy:
    args: [-y]
  claude:
    args: [--dangerously-skip-permissions]

# Custom agents with alternative binaries
custom_agents:
  - name: claude-dev
    type: claude
    bin: claude-dev
    data_dir: .claude-dev
    args: [--no-permissions-check]
